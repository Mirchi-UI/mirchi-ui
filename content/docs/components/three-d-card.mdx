## Usage

```tsx
import { ThreeDCard } from "@/components/modules/3d/three-d-card";

export function App() {
  return (
    <ThreeDCard 
      title="Interactive 3D" 
      description="This card renders a real 3D scene."
    />
  );
}
```

## Props

- `title?: string` - The title of the card.
- `description?: string` - The description text.
- `className?: string` - Additional classes for the container.
- `color?: string` - Color of the 3D mesh wireframe.

## Dependencies

This component requires the following dependencies:

```bash
pnpm add three @types/three @react-three/fiber @react-three/drei
```
