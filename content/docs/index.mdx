## For shadcn/ui Based Components

<div className="font-geist text-base leading-relaxed space-y-4 mb-6">
  <p>
    Some of our components are built on top of{" "}
    <span className="font-bold">shadcn/ui</span>. If you're using these
    components, you'll need the{" "}
    <code className="font-mono text-sm bg-muted px-1.5 py-0.5 rounded">cn</code>{" "}
    utility function.
  </p>
</div>

### Install the `cn` Utility

Add it via our CLI:

<Tabs items={["bun", "npx", "pnpm"]}>

```bash tab="bun"
bunx shadcn@latest add https://mirchi-ui.vercel.app/r/utils.json
```

```bash tab="npx"
npx shadcn@latest add https://mirchi-ui.vercel.app/r/utils.json
```

```bash tab="pnpm"
pnpx shadcn@latest add https://mirchi-ui.vercel.app/r/utils.json
```

</Tabs>

### Or Add It Manually

Create or update your `utils.ts` file:

<Tabs items={["bun", "npx", "pnpm"]}>

```bash tab="bun"
bunx install motion clsx tailwind-merge
```

```bash tab="npx"
npx install motion clsx tailwind-merge
```

```bash tab="pnpm"
pnpx add motion clsx tailwind-merge
```

</Tabs>

```tsx
import { type ClassValue, clsx } from "clsx";
import { twMerge } from "tailwind-merge";

export function cn(...inputs: ClassValue[]) {
  return twMerge(clsx(inputs));
}
```

    <strong>Note:</strong> You only need this if you're using components marked
    as "shadcn/ui Based". Standalone components work without any setup.
  </p>
