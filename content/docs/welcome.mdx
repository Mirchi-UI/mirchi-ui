## Motivation

<div className="my-8 font-geist p-6 rounded-lg bg-gradient-to-br from-purple-50 to-pink-50 dark:from-purple-950/20 dark:to-pink-950/20 border border-purple-200 dark:border-purple-800">
  <p className="text-base leading-relaxed">
    I've been building websites for years and always found myself searching through old files to reuse components. 
    It was annoying. So I thought—<strong>why not create a place where all my components live together?</strong>
  </p>

{" "}

<p className="text-base font-geist leading-relaxed mt-4">
  One of my goals was to contribute to component libraries. I discovered amazing
  projects like{" "}
  <a
    href="https://www.ui-layouts.com/get-started"
    target="_blank"
    rel="noopener noreferrer"
    className="text-purple-600 font-gMono dark:text-purple-400 hover:underline font-medium"
  >
    UI-Layouts
  </a>{" "}
  and other awesome UI libraries from Bangladesh. I forked them, learned how
  they worked, and now I'm building my own.
</p>

  <p className="text-base leading-relaxed mt-4">
    Libraries like <a href="https://ui.shadcn.com" target="_blank" rel="noopener noreferrer" className="text-purple-600 dark:text-purple-400 hover:underline">shadcn/ui</a> and{" "}
    <a href="https://ui.aceternity.com" target="_blank" rel="noopener noreferrer" className="text-purple-600 dark:text-purple-400 hover:underline">Aceternity UI</a> inspired me to create something free and open for everyone.
  </p>
