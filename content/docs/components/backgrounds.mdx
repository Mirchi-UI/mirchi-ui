## Background Circles

An animated background with rotating orbital circles and a glowing gradient effect. It supports multiple color variants.

## Usage

```tsx
import { BackgroundCircles } from "@/components/modules/background-circles";

export function Hero() {
  return (
    <div className="relative h-screen w-full overflow-hidden">
      <BackgroundCircles 
         title = "Build Something Amazing",
  description = "Create beautiful, animated interfaces with ease",
 
  variant = "octonary"
      />
    </div>
  );
}
```

## Props

The `BackgroundCircles` component accepts the following props:

- `variant`: Color theme (`primary`, `secondary`, `tertiary`, `quaternary`, `quinary`, `senary`, `septenary`, `octonary`).
- `title`: Main heading text.
- `description`: Subtext description.
- `className`: Additional CSS classes.

```tsx
interface BackgroundCirclesProps {
  title?: string;
  description?: string;
  className?: string;
  variant?: "primary" | "secondary" | "tertiary" | "quaternary" | "quinary" | "senary" | "septenary" | "octonary";
}
```
